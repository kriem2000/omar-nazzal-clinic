# Omar Nazzal Clinic Links Page

Static, serverless link page for Dr. Omar Ahmad Nazzal. It uses only:

- `index.html`
- `logo.png`
- `logo chapes.png`
- `assets/logo-main.jpg`
- `assets/logo-shapes.jpg`
- `assets/brand-mark.svg`
- `assets/brand-lines.svg`

There is no build step and no backend.

## Free Hosting Options

### Netlify Drop

Fastest option for this page.

1. Go to Netlify Drop.
2. Drag this folder into the upload area.
3. Netlify gives you a public `netlify.app` link.
4. Rename the site in Netlify settings if you want a cleaner URL.

### GitHub Pages

Best option if this folder will stay in a GitHub repository.

1. Push these files to GitHub.
2. Open repository settings.
3. Go to Pages.
4. Choose the branch and `/root` folder.
5. GitHub gives you a `github.io` link.

### Cloudflare Pages

Good option for a fast global static site.

1. Create a Pages project.
2. Use Direct Upload or connect the GitHub repository.
3. Set build command to empty.
4. Set output directory to `/`.

## QR Code Flow

After hosting, copy the public URL and generate a QR code for that URL. The QR code should point to the hosted page, not to each individual social link.
